# Trigger Checklist

Check on EVERY user message:

1. **Committing?** ("commit", "stage", "push") -> `pre-commit-check` BEFORE git op
2. **Corrected?** ("that's wrong", "actually...", "you forgot") -> `log-mistake`
3. **Session ending?** ("done", "thanks", "bye") -> `session-end-checklist`
4. **Solved something complex without KB?** (3+ files, 5+ exchanges) -> `document-solution`
5. **Modified KB files?** -> `check-kb-index`

Priority if multiple fire: 1 > 2 > 5 > 4 > 3
