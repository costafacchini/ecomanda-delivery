# Memory Compress

Compress memory/KB/CLAUDE.md files to reduce input tokens. Preserves all technical substance, code, URLs, and structure.

**Trigger**: `/memory-compress <filepath>`

Backs up the original as `<filename>.original.md` before overwriting. Works on `.md`, `.txt`, and other natural language files only — never touches code files.

See `SKILL.md` for the full skill definition used by AI agents.
